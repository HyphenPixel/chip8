# Another Chip8 Interpreter
This is just a simple chip8 interpreter that is written in pure C using Raylib for the rendering and sound.

### Build
A few prerequisites:
1. Raylib and any of it's required libs
2. Premake5
3. GCC or Clang

Build Instructions:
1. cd <project/dir>
2. premake5 gmake2
3. make config=<configuration>
